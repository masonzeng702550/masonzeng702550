## 用Python程式來玩POPCAT

這個方法不只能邊玩邊學，還可以不用每次玩都要自己複製程式碼或用壞鍵盤XD

### 步驟一

安裝Python(點→[me](https://www.python.org/downloads/)←，選擇與自己電腦作業系統的版本)

### 步驟二

首先開啟cmd ，並輸入：

```
pip install selenium
```

接著按下Enter來安裝Selenium

![](/images/c024cf68e91b/img02.png)

### 步驟三

首先在專案中使用from…import…的方式，匯入selenium套件。其中的 webdriver可以幫助我們透過selenium開啟各種不同的瀏覽器驅動程式，好讓我們可以操作該瀏覽器的行為

```
from selenium import webdriver
```

這個功能能夠驅動的瀏覽器包括IE、Firefox以及Chrome。在這裡我們用Chrome來進行製作

### 步驟四

由於selenium在Chrome 中操作必須透過Chrome WebDriver程式，所以要先安裝其驅動程式Chrome WebDriver。驅動程式可點→[me](https://chromedriver.chromium.org/)←下載符合自己作業系統和瀏覽器版本的Chrome WebDriver驅動程式

### 步驟五

並將變數"url"(中文即網址)設為https://popcat.click/

```
url='https://popcat.click/'
```

### 步驟六

並將driver以變數"url"的網址進行get

```
driver.get(url)
```

### 步驟七

接著我們從POPCAT網站按下右鍵打開檢查原始碼，我們觀察到次數的增加極有可能和其中的標籤"app"有關

![POPCAT 原始碼檢查(來源：https://popcat.click/)](/images/c024cf68e91b/img03.png)

### 步驟八

所以寫入此id讓他幫我們自動執行

```
cat=driver.find_element_by_id(‘app’)
```

### 步驟九

寫入for的程式執行10000次，下一行縮排後寫入cat.click來自動點擊

```
for i in range(10000):
```

```
  cat.click()
```

### 步驟十

讓我們來看看成果吧

![](/images/c024cf68e91b/img04.png)

### 完整程式

```
from selenium import webdriver
```

```
driver=webdriver.Chrome()
```

```
url='https://popcat.click/'
```

```
driver.get(url)
```

```
cat=driver.find_element_by_id('app')
```

```
for i in range(1000):
```

```
  cat.click()
```
